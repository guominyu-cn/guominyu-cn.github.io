## 文字格式

Markdown 支持多种文字格式：

- **粗体文字** - 用于强调重要内容
- *斜体文字* - 用于引用或轻微强调
- ~~删除线~~ - 表示已删除或不再相关的内容
- `行内代码` - 用于技术术语或代码片段

### Python 示例
```python
def fibonacci(n):
    """生成斐波那契数列"""
    a, b = 0, 1
    for _ in range(n):
        yield a
        a, b = b, a + b

# 使用示例
if __name__ == "__main__":
    print(list(fibonacci(10)))
